# custom-js-blocker

特定のWebサイトにおいて、ワンクリックでJavaScriptを自動的に無効化（ブロック）できるGoogle Chrome / Microsoft Edge用の拡張機能です。  
最新の **Manifest V3** に準拠しており、外部通信を一切行わない完全ローカル動作の安全な設計になっています。

過剰な広告、ポップアップ、追跡スクリプト（トラッカー）によるページの読み込み遅延や動作の重さを解消し、快適なブラウジングをサポートします。

---

## 導入方法（インストール）

本ツールは開発者モードを利用して手動で導入します。

### 1. パッケージ（ZIP）のダウンロード
以下のいずれかの方法で最新版のZIPファイルをダウンロードし、任意の場所に解凍してください。

* **公式ブログからダウンロード（推奨）**
   [ツール公開とか雑記](https://deve1operman.blogspot.com/2026/06/blog-post.html)
* **GitHubからダウンロード**
  このサイトから取得してください。

### 2. ブラウザへの読み込み
1. ChromeまたはEdgeを開き、拡張機能の管理画面（ `chrome://extensions/` または `edge://extensions/` ）にアクセスします。
2. 画面右上（Edgeは左下）にある「デベロッパー モード」をONにします。
3. 「パッケージ化されていない拡張機能を読み込む（展開して読み込む）」ボタンをクリックします。
4. 解凍したフォルダ（ `manifest.json` が入っているフォルダ）を選択すると、インストールが完了します。
